Live Trace Tool Instructions

To use the Azure SignalR Live Trace Tool:

  1. Go to Azure Portal → Your SignalR Service
  2. Navigate to "Live trace settings" in the left menu
  3. Enable live trace and configure trace levels
  4. Use the live trace viewer to see real-time connection and message data
  5. Filter by connection ID, user ID, or message type
